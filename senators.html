US Senators and Stock Trading

This was a data collection and visualization project using Python and the Selenium package. Financial disclosures can be found on the United States Senate Financial Disclosures website.

Project Inspiration

I became inspired to scrape financial disclosures at the beginning of the 2020 Coronavirus Pandemic after hearing news of mass stock selloffs from Senators. After learning about the STOCK Act, I became curious to see if I could create a dataset with all these dislosures. I settled on using Selenium and Python as a glorified copy-paste robot.

How it Works

If you've checked out the code, the nested loops might be a bit confusing. Here's an rundown on how it works in English:

open website;
select preferred options;

for every page:
	for every filing on the page:
		open filing
		copy-paste all details
		return to 'main' page
	hit the next button (and go to page n+1)

And voila, a couple thousand clicks and a few hours later and you've successfully scraped government property.
