About Serious SQL

Serious SQL is a hands on class where I am learning SQL by doing realistic case studies with sample databases. Upon finishing the initial course, I'll take a crack at the 8 Week SQL challenge, where one week represents a case study.

While some of the topics get pretty dense, the ones I am prioritizing are:

  • Exploratory Data Analysis
  • Table Joins
  • Common Table Expressions and Subqueries
  • To see a full list of topics, check them out here!

    • The 8 Week SQL Challenge

    The 8 Week SQL Challenge is a side 'quest' for Serious SQL and represents a summary of the skills that I learned throughout the course. They are set up with sample databases and act as case studies to push my SQL skills to the test. Of course, this is still in progress, so check in frequently to see my progress.
